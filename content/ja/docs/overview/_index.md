## 🔧事前準備

ワークショップを始める前に、お使いの端末のブラウザから[Azure OpenAI Studio](https://oai.azure.com/)にアクセス出来ることを確認します。

 [Azure OpenAI Studio](https://oai.azure.com/)
  ![](images/aoai-studio.png)


!> もし [Azure OpenAI Studio](https://oai.azure.com/)にアクセスできない場合は、 [OpenAI社 ChatGPT](https://www.bing.com/)を利用してワークショップを行うことも可能です。ただし、プロンプトに **絶対に機密情報を含めない** よう細心の注意してください。

Azure OpenAI Studio の **[Chat playground]** にプロンプトを入力し、送信します。
![](images/aoai-studio0.png)


チャットを削除したいときは、 **[Clear Chat]** ボタンをクリックします。
![](images/aoai-studio1.png)

あらかじめ事前情報与えずに、質問や指示のみを入力する方法です。プロンプトが短くて済みますが、複雑な質問に回答できない場合もあります。
これをZero-shot プロンプトと呼びます。なお「ショット」とはプロンプトにおいて例を意味します。


### :pen: 例題
次のプロンプトを、ChatGPTに入力します。

```text
自動車の購入に関するアンケートを作ってください。店舗や車種選びの基準や購入のきっかけなどの意見が知りたいです。
```

### :robot: 出力例

```md
もちろん、自動車の購入に関するアンケートを作成します。以下は質問の例です。必要に応じて質問を追加、修正、または削除して、調査に含める情報をカスタマイズしてください。

**自動車購入に関するアンケート**

**個人情報（オプション）:**
1. お名前（任意）:
2. 年齢（任意）:
3. 性別（任意）:

**1. 自動車購入の動機:**
4. 自動車を購入しようと思ったきっかけは何ですか？（複数選択可）
   - [ ] 交通手段として必要
   - [ ] 既存の車の更新
   - [ ] 趣味や娯楽
   - [ ] 仕事に関連
   - [ ] 環境への配慮（電気自動車など）
   - [ ] その他（具体的に記述してください）:

**2. 自動車購入の予算:**
5. 自動車購入にどの程度の予算を割けると考えていますか？

   - [ ] 100万円未満
   - [ ] 100万円から300万円
   - [ ] 300万円から500万円
   - [ ] 500万円から1000万円
   - [ ] 1000万円以上

**3. 自動車の選択基準:**
6. 自動車を選ぶ際に、どのような基準が重要でしたか？（複数選択可）
   - [ ] ブランドやメーカー
   - [ ] 燃費の良さ
   - [ ] サイズや車種
   - [ ] 安全性
   - [ ] テクノロジー機能（ナビゲーション、自動運転、エンターテイメントなど）
   - [ ] 販売店の評判や評価
   - [ ] 環境への配慮
   - [ ] 価格
   - [ ] その他（具体的に記述してください）:

**4. 自動車の車種選び:**
7. どの車種を選びましたか？具体的な車種名を教えてください。

**5. 自動車の評価:**
8. 購入した自動車について、以下の点で評価をしていただけますか？（1から5までのスケールで評価してください）
   - [ ] 燃費の良さ: 1  2  3  4  5
   - [ ] 安全性: 1  2  3  4  5
   - [ ] 快適性: 1  2  3  4  5
   - [ ] テクノロジー機能: 1  2  3  4  5
   - [ ] デザインと外観: 1  2  3  4  5

**6. 自動車関連の意見や提案:**
9. 自動車購入に関するあなたの意見や提案があれば、お聞かせください。

**7. その他のコメント（オプション）:**
10. その他、自動車に関するコメントや感想があれば、ご自由にお書きください。

アンケートのデザインや質問の順序を調整して、調査対象者に適した形式にカスタマイズできます。このアンケートを利用して、自動車購入に関する価値ある洞察を収集できることを願っています。
```

**GitHub Copilot** は、**既存のコードに自動でコメントを生成**してくれる機能を持っています。コードロジックの説明が不足している場合でも、手軽に解説文を追加できるのが大きなメリットです。ここでは、**「階乗 (factorial)」** のサンプルコードを使い、Copilot でどのようにコメントを補うか、具体的な流れを見てみましょう。

---

### :pen: 例題

まずは、**コメントのない** `factorial` メソッドを **Java** で用意します。

```java
// このメソッドの処理内容を日本語で説明するコメントを追加してください
public class FactorialExample {
    public static int factorial(int n) {
        if (n <= 1) {
            return 1;
        }
        return n * factorial(n - 1);
    }
}
```

ここでは**アルゴリズム**についての説明が不足しています。
この状態でCopilot に「コメントを自動生成して」と促すと、**メソッドの説明**などを自動的に追加してくれます。

---

### :robot: 出力例

Copilot が提案するコメントのイメージです。実際の結果はバージョンや文脈によって異なる可能性があります。

```java
// このメソッドの処理内容を日本語で説明するコメントを追加してください
// このメソッドは、再帰的に階乗を計算するメソッドです。
// 階乗とは、n! = n * (n-1) * (n-2) * ... * 1 のように、nから1までの整数を掛け合わせた値を表します。
// このメソッドは、nが1以下の場合は1を返し、それ以外の場合はn * factorial(n - 1)を返します。
// この再帰的な処理により、nから1までの整数を掛け合わせた値を求めることができます。

public class FactorialExample {
    public static int factorial(int n) {
        if (n <= 1) {
            return 1;
        }
        return n * factorial(n - 1);
    }
}
```

**先頭にコメント生成の指示を書く**（例：`// このメソッドの処理内容を日本語で説明するコメントを追加してください`）と、Copilot がコード上部や行間に説明文を挿入してくれるはずです。

---

### :memo: 練習

1. **別のコメント形式を使う**  
   - 例：複数行コメント (`/* ... */`) で「このメソッドの入出力やアルゴリズムを詳しく説明してほしい」と記載  
2. **他の関数に適用してみる**  
   - 階乗以外のアルゴリズム（フィボナッチ数列など）に試す → どのような説明文が付くかを観察

---

## まとめ

- **既存コード**に**自動でコメントを生成** → Copilot がロジックを推測し、**説明文**を追加  
- **短い指示**（「このメソッドを日本語で解説して」）を**コメント**に書くだけで、**解説文**を自動提案  
- **アルゴリズムの解説**や**変数の役割**を短時間で補えるため、**初学者や他メンバー**が理解しやすくなる

あらかじめ事前情報与えずに、質問や指示のみを入力する方法です。プロンプトが短くて済みますが、複雑な質問に回答できない場合もあります。
これをZero-shot プロンプトと呼びます。なお「ショット」とはプロンプトにおいて例を意味します。


### :pen: 例題
次のプロンプトを入力します。

```text
消費者の購買意向に関するアンケートを作成してください。商品の選び方や購入の動機、購入場所などについての意見を知りたいです。
```

### :robot: 出力例

```md
アンケートの作成には、以下のような質問項目を含めることができます。参考までにご覧ください。  
   
1. あなたの性別は何ですか？  
- 男性  
- 女性  
- 非バイナリー  
   
2. あなたの年齢は何歳ですか？  
- 18歳未満  
- 18歳〜24歳  
- 25歳〜34歳  
- 35歳〜44歳  
- 45歳〜54歳  
- 55歳以上  
   
3. 商品を選ぶ際に重視する要素を教えてください。複数選択可。  
- 価格  
- 品質  
- ブランド  
- デザイン  
- 機能性  
- 口コミ・評価  
- その他（自由記述）  
   
4. 商品を購入する動機は何ですか？複数選択可。  
- 自己満足  
- 必要なものを購入するため  
- インスタグラムやSNSでの写真映え  
- 他の人と差をつけるため  
- 最新トレンドに追いつくため  
- その他（自由記述）  
   
5. 商品を購入する場所はどこですか？複数選択可。  
- デパート・百貨店  
- 専門店  
- オンラインショップ（Amazon、楽天など）  
- フリーマーケット  
- ブランド直営店  
- その他（自由記述）  
   
6. 商品を購入する前に、どのような情報収集を行いますか？複数選択可。  
- インターネット検索  
- 口コミ・評価サイトの閲覧  
- 友人・家族の意見を聞く  
- 店舗での実物確認  
- 店員への相談  
- その他（自由記述）  
   
7. 商品の購入時に重要視するサービスや特典はありますか？複数選択可。  
- 割引価格  
- ポイント還元  
- ギフトラッピング  
- 返品・交換のしやすさ  
- 保証・アフターサービス  
- その他（自由記述）  
   
8. 今後の購買意向に関して、どのような要素が影響すると思いますか？複数選択可。  
- 価格の変動  
- 新商品の登場  
- ブランドイメージ  
- 口コミ・評価  
- 広告・宣伝  
- その他（自由記述）  
   
以上が一般的な消費者の購買意向に関するアンケートの例です。必要に応じて質問項目や選択肢を追加・変更して、自社や製品に合ったアンケートを作成してください。
```

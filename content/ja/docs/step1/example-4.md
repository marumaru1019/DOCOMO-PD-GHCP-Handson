次のようにいくつかの類似した質問をしたあとに、モデルに回答させたい問いを書きます。
これをFew-shot プロンプトと呼びます。Few-shot プロンプトとはあらかじめ例を提示することで、タスクを実行するための方法です。

### :pen: 例題

```text
以下は有名な映画です。

映画名: スター・ウォーズ
ジャンル: ファンタジー
映画名: ハリー・ポッター
ジャンル: ファンタジー
映画名: ミッション：インポッシブル
ジャンル: アクション
映画名: ジュラシック・パーク
ジャンル: アドベンチャー
映画名: ラ・ラ・ランド
ジャンル: ミュージカル

映画名: パイレーツ・オブ・カリビアン
```

### :robot: 出力例

```text
ジャンル: アクション、冒険
```

### :pen: 例題
次の文章は、映画を見に来たお客様のアンケートの自由記入欄のコメントを分析したものです。

次のプロンプトを入力します。

```text
ハリーポッターは最高だ! // ポジティブ
君たちはどう生きるかは難しい!// ネガティブ
ドラえもんは感動する! // ポジティブ
...

インディージョーンズはワクワクする! //

```

### :memo: 練習
